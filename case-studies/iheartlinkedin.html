iHeartLinkedIn

An AI content engine for LinkedIn that takes you from blank page to scheduled post in under a minute. Built, launched, and grown solo.

10K+ generations
40+ countries
1 person team

Problem

Professionals know LinkedIn drives careers, but consistent posting is a grind. Writing takes an hour per post, most drafts never ship, and generic AI tools produce content that reads like everyone else's.

Solution

A focused tool that learns a user's voice from their past posts, drafts in that voice, and handles scheduling. One input, one output, no dashboard sprawl. Shipped an MVP in three weeks, then iterated weekly from usage data.

Tech stack

Next.js TypeScript OpenAI API Supabase PostgreSQL Vercel

Key decisions

Start with the narrowest useful workflow

The first version only needed to turn a prompt into a usable LinkedIn post. Keeping the loop small made the product easy to test, ship, and improve from real usage.

Optimize for voice, not volume

Most writing tools produce more text. This one focused on preserving user voice from prior posts, which made the output feel publishable instead of generic.

Measure the path to published content

The key product metric was not signups or generations alone. It was how quickly a user could move from an idea to something ready to schedule.

Results

Organic launch, zero ad spend. Word of mouth carried it across 40+ countries; retention held because the output actually sounds like the user.
